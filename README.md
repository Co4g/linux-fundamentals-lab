# linux-fundamentals-lab
Repositório dedicado ao estudo prático de Linux, explorando fundamentos do sistema, redes e conceitos de segurança. Todo o conteúdo é documentado a partir de um laboratório pessoal e evolui conforme o aprendizado.

Todo o conteúdo aqui documentado é fruto de um laboratório pessoal, criado com
o objetivo de consolidar conhecimento técnico através da prática e da
documentação contínua.

## Objetivos

- Aprender e consolidar fundamentos do Linux
- Entender como o sistema se comporta na prática
- Relacionar Linux com redes e segurança
- Desenvolver raciocínio técnico e capacidade de análise
- Criar um histórico de aprendizado organizado e evolutivo

## Ambiente

- Sistema operacional: Linux Mint (base Ubuntu)
- Tipo de ambiente: laboratório local
- Foco principal: uso de terminal e análise do sistema

## Estrutura do repositório

- `week1-filesystem-permissions/`  
  Estudo sobre sistema de arquivos, usuários, grupos e permissões.

Novas pastas e conteúdos serão adicionados conforme o avanço dos estudos.

## Observação

Este repositório é evolutivo. O conteúdo pode ser revisado, expandido ou
corrigido à medida que novos conceitos forem aprendidos.
